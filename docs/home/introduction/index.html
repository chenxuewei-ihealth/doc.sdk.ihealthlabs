iHealthlabs SDK

Introduction#

iHealth SDK is a API for connection and managing iHealth devices in mobile applications, and help you easy to connect and operate iHealth devices, integrate iHealth SDK into your applications rapidly.

Easy to Measurement

iHealth SDK was designed for connecting iHealth devices easily and use it quickly.

Focus on What Matters

iHealth SDK lets you focus on your app, and don't to know the device low-level interaction.

Support Multi-platform

iHealth SDK support multi-platform Android iOS and React Native.
